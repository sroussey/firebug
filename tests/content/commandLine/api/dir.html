Command Line API: dir

This test-case is intended for dir method.
Jan Odvarko, odvarko@gmail.com

  1. Open Firebug and enable the Console panel.
  2. Execute the following expression in the command line:
    dir({'a': {'b': {'c': 1}}});
  3. the console panel should display a log (one row) with variable a and value Object { b={...} }.
  4. Expand the object a, expand the object b, collapse b and collapse a. All should work as expected.
